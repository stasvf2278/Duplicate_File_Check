Project Name:

Duplicate File Check

Description:

This script copies the names of all the files in two folders as two seperate lists, removes the extensions of the file names in these lists and then compares the extensionless file names of the two folders. Duplicate file (one in each folder) have their names printed.

Usage:

Note: Written and tested in Pyton 3.7

Note: This script was written for Windows.

Installation:

This script is written in the main() feature.
